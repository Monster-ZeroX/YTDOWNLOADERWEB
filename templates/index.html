Getting a "Sign in" or "Confirm you're not a bot" error?

YouTube sometimes requires you to be logged in. You can use a cookies file from your browser to fix this.

  1. Install the Get cookies.txt LOCALLY extension for Chrome.
  2. Go to YouTube and log in.
  3. Click the extension icon and then "Export".
  4. Save the `youtube.com_cookies.txt` file.
  5. Upload that file above before clicking "Get Video".
